# Settings
## autocomplete
- Windows menu → Preferences → Java → Editor → Content assist
- Change the default in Auto activation triggers for Java to :
```txt
._abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ
```
