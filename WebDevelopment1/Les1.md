# Basis

```html
<p>Dit is een parafgraaf</p>
<!-- <p> is een begin tag -->
<!-- </p> is een eind tags -->
<!-- <br> is ook een tag -->

<!-- <p>Dit is een parafgraaf</p> is een element -->

<p class="editor-note">Dit is een voorbeeld van een klasse</p>
<!-- class is een atrribuutnaam en editor-note is de attribuutwaarde, samen is dit een attribuut -->




```


```html

<html> <!-- is begin van je html document -->

  

<head> <!-- is begin van je metadata (informatie over data) -->

    <title>Mijn demo</title><!-- titel van pagina -->

</head>

  

<body> <!-- is begin van je paginainhoud, bevat de data -->

    <!-- Een return of meerdere spaties wordt herleid naar een spatie -->

    <!-- <p></p> is een paragraaf -->

    <!-- <br> is volgende lijn -->

    <p>Dit is <br> een parafgraaf</p>

    <!-- <strong></strong> is in vet zetten -->

    <p>Mij Cat is <strong>very</strong> grumpy</p>

</body>

  

</html> <!-- is einde van je html document -->

```
